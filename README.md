# Lámpara con sensor crepuscular

A través de una fotocelda que detecta los niveles de luz se controla un relevador con una carga para encender o apagar una lámpara según el nivel detectado con un control de sensibilidad para personalizar el punto en el que la lampara se encienda.

## Características

- Automatización sin necesidad de programación
- Diseño de PCB especial para el proyecto
- Control de setpoint directo en el proyecto

## Estructura del repositorio
**Docs:** Contiene fotos y videos más detallados del funcionamiento del proyecto
**Hardware:** Contiene los archivos necesarios para replicar el proyecto, esquemáticos, archivos gerber, BOM (Bill of materialls) y POS (Archivos de posición para el pedido de la PCB)

## Cómo empezar

En caso de no tener la PCB del proyecto siga las conexiones indicadas del esquemático en una protoboard teniendo precaución extra en la parte de potencia de la lampara para evitar accidentes, se recomienda hacer una prueba con un elemento simple que muestre su funcionamiento, por ejemplo un LED.

En caso de tener la PCB del proyecto, solo hace falta soldar según la asignación del componente que se indica en la misma placa por el reverso.
